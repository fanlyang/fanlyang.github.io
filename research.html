Research Experience

Molecular Dynamics Parallelization

Developed MPI-based domain decomposition for molecular dynamics with cell-list acceleration and efficient force communication. Implemented a scalable C++ MD solver using ghost-cell updates and non-blocking MPI communication. Achieved near-linear scaling up to 512 cores on HPC systems.

Quantum–Classical Hybrid Solvers

Implemented HHL and VQLS algorithms to solve linear systems using hybrid quantum-classical architectures. Explored hybrid solvers via the XACC framework, focusing on noise mitigation and circuit depth optimization for NISQ devices.

Nonlinear FEM Solver in Java

Developed an object-oriented finite element code for nonlinear mechanics using Newton–Raphson and arc-length methods. Implemented modular material and boundary classes, achieving reliable convergence for complex post-buckling simulations.

AI for Scientific Computing (to be updated)

Integrating surrogate models with traditional solvers to accelerate Uncertainty Quantification (UQ) workflows. Leveraging Gaussian Processes and Neural Operators for PDE approximation and accelerating multiscale simulations.
